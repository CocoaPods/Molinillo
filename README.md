A generic dependency-resolution implementation.

## Installation

Add this line to your application's Gemfile:

    gem 'Resolver', :git => 'https://github.com/CocoaPods/Resolver'

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install Resolver
